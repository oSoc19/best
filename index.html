This is Open Summer of code 2016

Open Summer of Code is a 4 week programme in different Belgian cities during the summer of 2016. The goal: provide Belgian based students the training, network and support necessary to transform open innovation projects into powerful real-world services.

Students

We offer Belgian students the chance to learn and work in a professional work environment, so that they can grow as a young professional. These are the profiles we need:

  • Front-end developer

    HTML, CSS, JS, ...

  • Back-end developer

    PHP, JS, Java, ...

  • Graphic designer

    Photoshop, Illustrator, ...

  • User experience designer

    UX, Interaction Design, ...

  • Communication/marketing manager

    Social Media, Communication Strategy, ...

  • Business developer

    Business Modeling, Strategy, ...
